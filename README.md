# 🚗 Cars Dealership - Full-Stack Application

## Description
Application web complète pour un concessionnaire automobile national. Permet de visualiser les concessionnaires, consulter et soumettre des avis avec analyse de sentiment.

## Technologies
- Backend: Django, Django REST Framework
- Frontend: React.js
- Base de données: SQLite / MongoDB
- Analyse de sentiment: NLP
- Déploiement: Docker, Kubernetes, IBM Cloud Code Engine
- CI/CD: GitHub Actions

## Installation
```bash
git clone https://github.com/ju-info-19/cars-dealership.git
cd cars-dealership/server
