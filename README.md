# 🚗 Cars Dealership - Projet Full-Stack

## Nom du projet
**Cars Dealership**

## Description
Application web complète pour un concessionnaire automobile national.

## Technologies
- Backend: Django, Django REST Framework
- Frontend: React.js
- Base de données: SQLite
- Déploiement: IBM Cloud Code Engine
- CI/CD: GitHub Actions

## Installation
```bash
git clone https://github.com/ju-info-19/cars-dealership.git
cd cars-dealership
pip install -r requirements.txt
python manage.py runserver
